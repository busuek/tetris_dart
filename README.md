# Разработка игрового приложения Тетрис Cloud с интеграцией Firebase.
## Полное Техническое задание

### Цель задания
Создать мобильное приложение Tetris.
Требования:
1. Игровое поле - реализовать сетку 10x20, где каждая ячейка - это виджет, меняющий цвет, в зависимости от состояния (пусто\занято).
2. Логика JSON. Описать состояние падающей фигуры и игрового поляв виде обьектов, готовых к передаче в сеть.

Синхронизация > Подключить Firebase для сохранения рекордов и возможности "паузы" игры с сохранением состояния в облаке.

### Инструкция к выполнению:

Шаг 1. Настройка окружения и Firebase
- В терминале выполните команду flutterfire configure, чтобы связать проект с вашей БД.
- Добавьте зависимости  firebase_core и firebase_database.

Шаг 2. Создание модели и парсинга
- Создайте класс TetrisGame с методом factory TetrisGame.from.Json. Это позволит
 приложению восстанавливать игру, загружая JSON-дерево из Firebase

Шаг 3. Разработка интерфейса и событий
- События. Используйте IconButton для упраавления (Влево, Вправо, Поворот). Каждое нажатие - это событие, меняющее координаты фигуры через setState.
- Подсказки. Добавьте Tooltip для каждой кнопки управления, чтобы обьяснить игроку их функции.
- Индикация. Реализуйте изменения цвета текста счета: если текущий счет больше рекорда из Firebase, цвет становится золотым.

Шаг 4. Навигация и передача данных
- При наступлении события Game Over, используйте Navigator.push для перехода на экран "Итоги". Передайте туда обьект с финальной статистикй для записи в Firebase.
